# 使い方

## 基本的な使い方

### 1. ツールを開く

メニューから `Tools > Gradation Baker` を選択してウィンドウを開きます。

### 2. メッシュを追加する

対象メッシュセクションのドロップエリアに、HierarchyからSkinnedMeshRendererまたはMeshRendererをドラッグ＆ドロップします。複数のメッシュを追加できます。

各メッシュの左側にある「▶」アイコンを展開すると、以下の詳細設定を行えます：

- **UVチャンネル**: ベイク先のUVチャンネル（UV0〜UV3）
- **マスクテクスチャ**: テクスチャのRチャンネルをマスクとして使用（任意）
- **頂点カラーを使用**: 頂点カラーのRチャンネルをマスクとして使用
- **マスク反転**: マスクの適用を反転

### 3. グラデーションを設定する

「グラデーション」セクションでカラーを設定します。グラデーションエディタをクリックして色を編集できます。

### 4. グラデーション範囲を調整する

シーンビューでボックスハンドルを操作してグラデーションの範囲と方向を調整します：

- **回転ハンドル**: グラデーションの方向を変更
- **移動ハンドル**: ボックス全体を移動
- **赤いコーン（下）**: グラデーションの開始位置（0%）
- **緑のコーン（上）**: グラデーションの終了位置（100%）

「メッシュに合わせる」ボタンでメッシュのバウンディングボックスに自動フィットします。

### 5. ベイク＆保存

「ベイク＆保存」ボタンをクリックしてテクスチャを生成・保存します。
各メッシュごとに設定されたUVチャンネルとマスク設定が適用されます。

---

## 詳細機能

### 作業メッシュ

ModularAvatar等の非破壊スケールコンポーネント使用時に座標がズレる問題を回避するため、作業用の複製メッシュを生成できます。

- 「作業メッシュ生成」ボタンで全メッシュの作業用コピーを作成
- 作業メッシュ生成時に自動で「メッシュに合わせる」が実行されます

### グラデーションの保存/読込

「保存」ボタンでグラデーションをテクスチャとして保存し、「読込」ボタンで復元できます。保存先は `Assets/GeneratedGradation/gradation/` です。

### ミラー機能

「ミラー」セクションで軸を選択すると、グラデーションが指定軸で対称にミラー適用されます。シアン色のボックスがミラー位置を示します。

### 出力設定

| 設定 | 説明 |
|------|------|
| 解像度 | 出力テクスチャの解像度 |
| マテリアルのテクスチャフォルダに出力 | 各メッシュのマテリアルのテクスチャと同じフォルダに出力（初期設定でON） |
| 保存先 | 上記がOFFの場合の保存先フォルダ |
| 背景色 | 透明/白/黒から選択 |

> **Note**: 保存完了後、自動的にProjectタブで最初に保存されたテクスチャが選択・表示されます。

### 言語切り替え

ウィンドウ右上の「Enable English Mode」チェックボックスで日本語/英語を切り替えられます。
 
 ## 注意事項
 
 - プレビューの重ねた表示は簡易的なものであり、生成されるテクスチャの色は透明度を1にした時の色になります（プレビュー用不透明度スライダーの値は出力に影響しません）。
